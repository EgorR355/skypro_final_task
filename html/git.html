Git

Настройки и создание файлов.

  • git init — Инициализация нового репозитория в текущей папке.
  • git clone — Клонирование существующего удаленного репозитория на локальный компьютер.
  • git config — Настройка параметров пользователя (имя, email).

Работа с изминениями

  • git status — проверка состояния файлов (измененные, отслеживаемые, новые).
  • git add . — добавление изменений в область подготовленных файлов (индекс) перед сохранением.
  • git commit -m "Комментарий" — фиксация (сохранение) изменений с описанием в истории.
  • git diff — просмотр конкретных изменений в коде до их добавления в индекс.

Ветвление и слияние

  • git branch — просмотр всех локальных веток в репозитории.
  • git checkout <имя_ветки> или git switch <имя_ветки> — переключение на указанную ветку.
  • git checkout -b <имя_ветки> — создание новой ветки и мгновенное переключение на неё.
